# 音声タイマー 仕様書

## 概要

指定した時刻に音声・動画ファイルを自動再生するタイマーアプリ。  
作業用BGMの再生や背景画像・動画の設定も可能。  
ブラウザ版（PWA）・デスクトップ版（Mac）の2形態で提供。

---

## 対象ユーザー

- 決まった時刻にアラーム音を鳴らしたいユーザー
- 配信・録音・動画制作などで決まったタイミングに音を出したい制作者
- 作業中にBGMを流しながらタイマーを管理したいユーザー

---

## 機能仕様

### タイマー機能

| 機能 | 詳細 |
|------|------|
| 時刻指定 | HH:MM:SS 単位で再生時刻を指定 |
| アラーム音（必須） | ローカルの音声・動画ファイルを選択して指定時刻に再生 |
| 複数タイマー | 異なる時刻・音声の組み合わせで複数登録可能 |
| ボリューム調整 | 0〜100% のスライダーで音量設定 |
| ループ再生 | トグルスイッチでループのON/OFF |
| カウントダウン | 最も近いタイマーまでの残り時間をリアルタイム表示 |
| タイマー削除 | スケジュール一覧から個別削除 |
| 全停止 | 再生中の音声停止＋全タイマーのクリア |

### アラームポップアップ

時刻になると画面中央にモーダルダイアログが表示される。

| 仕様 | 詳細 |
|------|------|
| 表示位置 | 画面中央固定 |
| 背景 | 半透明＋ブラーエフェクト |
| 表示内容 | 設定時刻・ファイル名 |
| アニメーション | ポップイン（スケール＋フェード） |
| 解除 | 「停止」ボタンをタップ／クリック |
| BGM | アラーム発火中も継続再生 |

### カウントダウンの色変化

| 残り時間 | 色 |
|----------|----|
| 60秒超 | 緑（テーマ依存） |
| 60秒以内 | 黄（テーマ依存） |
| 10秒以内 | 赤（テーマ依存） |

---

### 作業用BGM（オプション）

タイマー音とは独立して動作するBGMプレイヤー。

| 機能 | 詳細 |
|------|------|
| ファイル選択 | ローカルの音声・動画ファイルに対応 |
| 音量調整 | タイマー音量とは独立したスライダー |
| ループ再生 | トグルスイッチでON/OFF（デフォルトON） |
| 再生 / 一時停止 | ボタン1つで切り替え |
| 停止 | 再生位置を先頭に戻して停止 |

---

### 背景メディア（オプション）

アプリの背景に画像または動画を設定できる。

| 機能 | 詳細 |
|------|------|
| 画像対応 | JPEG / PNG / GIF など一般的な画像形式 |
| 動画対応 | MP4 など。**ミュート＋ループ**で自動再生 |
| 透明度調整 | 0〜100% のスライダーで調整（設定は `localStorage` に保存） |
| 削除 | 「✕ 削除」ボタンで背景をクリア |
| 画像・動画の切り替え | どちらか一方のみ表示（選び直すと自動で切り替え） |

---

### テーマ切り替え

| テーマ名 | フォント | カラー | 特徴 |
|----------|----------|--------|------|
| シンプル | システムフォント | 黒・白 | デフォルト。ミニマルなダークUI |
| メルヘン | DotGothic16（ドット文字） | ピンク・白 | 可愛らしいピクセル風デザイン |
| ナチュラル | Yomogi（手書き） | ベージュ・薄緑 | 温かみのある手書き風デザイン |

- 画面上部のボタンで即時切り替え
- 選択したテーマは `localStorage` に保存され、次回起動時も維持される
- フォントは Google Fonts から読み込み

```
┌─────────────────────────────┐

├─────────────────────────────┤

├─────────────────────────────┤

├─────────────────────────────┤

├─────────────────────────────┤

├─────────────────────────────┤
│  ─── オプション ───            │

└─────────────────────────────┘

※時刻到達時：画面中央にアラームポップアップを表示
```

## 技術仕様

| 項目 | 内容 |
|------|------|
| 動作環境 | モダンブラウザ（Chrome / Safari / Firefox / Edge） |
| 実装 | HTML + CSS + JavaScript（単一ファイル） |
| 依存ライブラリ | Google Fonts（DotGothic16 / Yomogi） |
| タイマー音生成 | Web Audio API（フォールバック用ビープ） |
| ファイル読み込み | `URL.createObjectURL()` によるローカルファイル参照 |
| タイマー精度 | `setInterval` 1秒間隔（±1秒の誤差あり） |
| データ永続化 | テーマ設定・背景透明度を `localStorage` に保存 |
| モバイル自動再生対策 | ファイル選択時・タイマー追加時に `play()→pause()` で再生権限を事前取得 |

## スマホ版（PWA）

Progressive Web App として提供。Safari の「ホーム画面に追加」でネイティブアプリに近い体験が得られる。

### インストール手順（iPhone）

1. Safari で公開URLを開く  
   `https://goldfishstar0055.github.io/audio-timer/audio-timer.html`
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択 → 「追加」

### PWA 構成ファイル

| ファイル | 役割 |
|---------|------|
| `manifest.json` | アプリ名・アイコン・表示モードの定義 |
| `sw.js` | Service Worker（オフラインキャッシュ） |
| `icons/icon-192.png` | ホーム画面アイコン（192×192） |
| `icons/icon-512.png` | スプラッシュ画面用アイコン（512×512） |
| `icons/apple-touch-icon.png` | iOS 向けアイコン（180×180） |

### PWA の制約

- **バックグラウンド時は音声再生が停止する**（iOS / Android の制限）
- タブを開いたまま・画面を点けたまま使用することで回避可能

## デスクトップアプリ版（Mac）

Electron を使用した Mac 向けデスクトップアプリ版。

### ブラウザ版との違い

| 項目 | ブラウザ / PWA版 | デスクトップ版 |
|------|----------------|--------------|
| バックグラウンド動作 | ✕ | ✅ ウィンドウを閉じてもタスクトレイで動作 |
| PCスリープ後の動作 | △ | ✅ スリープ復帰後も継続 |
| 自動再生制限 | あり | なし |

### 配布ファイル

| ファイル | 対象 |
|---------|------|
| `Audio Timer-1.0.0-arm64.dmg` | Apple Silicon Mac（M1/M2/M3/M4）|
| `Audio Timer-1.0.0.dmg` | Intel Mac |

### インストール手順

1. お使いの Mac に合った `.dmg` をダブルクリック
2. `Audio Timer` を `アプリケーション` フォルダにドラッグ
3. アプリケーションフォルダから起動

> **初回起動時の注意:** 「開発元を確認できません」と表示された場合は、システム設定 → プライバシーとセキュリティ → 「このまま開く」をクリック

---

## 既知の制約・注意事項

- **ブラウザ / PWA版はタブを閉じるとタイマー・BGMが停止する**
- **スマホでバックグラウンドになると音声が停止する**（OS制限）
- 背景画像・BGM・タイマー音のファイルはリロードで消去される（設定の永続化は未実装）
- ローカルファイルを `file://` で開いた場合、セキュリティ制限が適用される場合あり（GitHub Pages 経由推奨）
