# 音声タイマー 仕様書

## 概要

指定した時刻に音声・動画ファイルを自動再生するブラウザベースのタイマーアプリ。  
作業用BGMの再生や背景画像・動画の設定も可能。  
インストール不要。HTMLファイルをブラウザで開くだけで動作する。

---

## 対象ユーザー

- 決まった時刻にBGMやアラーム音を鳴らしたいユーザー
- 配信・録音・動画制作などで決まったタイミングに音を出したい制作者
- 作業中にBGMを流しながらタイマーを管理したいユーザー

---

## 機能仕様

### タイマー機能

| 機能 | 詳細 |
|------|------|
| 時刻指定 | HH:MM:SS 単位で再生時刻を指定 |
| 音声ファイル再生 | ローカルの音声・動画ファイルを選択して指定時刻に再生 |
| フォールバック音 | ファイル未選択時はビープ音（Web Audio API 生成）を使用 |
| ボリューム調整 | 0〜100% のスライダーで音量設定 |
| ループ再生 | トグルスイッチでループのON/OFF |
| 複数タイマー | 異なる時刻・音声の組み合わせで複数登録可能 |
| カウントダウン | 最も近いタイマーまでの残り時間をリアルタイム表示 |
| アラームUI | 時刻になるとオーバーレイ表示・「停止」ボタンで解除 |
| タイマー削除 | スケジュール一覧から個別削除 |
| 全停止 | 再生中の音声停止＋全タイマーのクリア |

### カウントダウンの色変化

| 残り時間 | 色 |
|----------|----|
| 60秒超 | 緑（テーマ依存） |
| 60秒以内 | 黄（テーマ依存） |
| 10秒以内 | 赤（テーマ依存） |

---

### 作業用BGM

タイマー音とは独立して動作するBGMプレイヤー。

| 機能 | 詳細 |
|------|------|
| ファイル選択 | ローカルの音声・動画ファイルに対応 |
| 音量調整 | タイマー音量とは独立したスライダー |
| ループ再生 | トグルスイッチでON/OFF（デフォルトON） |
| 再生 / 一時停止 | ボタン1つで切り替え |
| 停止 | 再生位置を先頭に戻して停止 |

---

### 背景メディア

アプリの背景に画像または動画を設定できる。

| 機能 | 詳細 |
|------|------|
| 画像対応 | JPEG / PNG / GIF など一般的な画像形式 |
| 動画対応 | MP4 など。**ミュート＋ループ**で自動再生 |
| 透明度調整 | 0〜100% のスライダーで調整（設定はlocalStorageに保存） |
| 削除 | 「✕ 削除」ボタンで背景をクリア |
| 画像・動画の切り替え | どちらか一方のみ表示（選び直すと自動で切り替え） |

---

### テーマ切り替え

| テーマ名 | フォント | カラー | 特徴 |
|----------|----------|--------|------|
| シンプル | システムフォント | 黒・白 | デフォルト。ミニマルなダークUI |
| メルヘン | DotGothic16（ドット文字） | ピンク・白 | 可愛らしいピクセル風デザイン |
| ナチュラル | Yomogi（手書き） | ベージュ・薄緑 | 温かみのある手書き風デザイン |

- 画面上部のボタンで即時切り替え
- 選択したテーマは `localStorage` に保存され、次回起動時も維持される
- フォントは Google Fonts から読み込み

---

## 技術仕様

| 項目 | 内容 |
|------|------|
| 動作環境 | モダンブラウザ（Chrome / Safari / Firefox / Edge） |
| 実装 | HTML + CSS + JavaScript（単一ファイル） |
| 依存ライブラリ | Google Fonts（DotGothic16 / Yomogi） |
| タイマー音生成 | Web Audio API（フォールバック用ビープ） |
| ファイル読み込み | `URL.createObjectURL()` によるローカルファイル参照 |
| タイマー精度 | `setInterval` 1秒間隔（±1秒の誤差あり） |
| データ永続化 | テーマ設定・背景透明度を `localStorage` に保存（タイマー・BGM・背景はリロードで初期化） |

```
┌─────────────────────────────┐
│  [シンプル] [メルヘン] [ナチュラル] │  ← テーマ切り替え
├─────────────────────────────┤
│  現在時刻（リアルタイム）        │
├─────────────────────────────┤

│  背景画像 / 動画 ＋ 透明度      │  ← 背景メディア

│  作業用BGM ＋ 音量 ＋ 操作     │  ← BGMプレイヤー
├─────────────────────────────┤
│  音声ファイル選択               │  ← タイマー音
├─────────────────────────────┤
│  ボリューム / ループ設定         │
├─────────────────────────────┤
│  カウントダウン表示              │

└─────────────────────────────┘

```

## 既知の制約・注意事項

- **ブラウザのタブを閉じるとタイマー・BGMは停止する**
- **PCをスリープするとタイマーが機能しない場合がある**
- ブラウザの自動再生ポリシーにより、ユーザー操作前は音声が再生されないことがある（タイマー追加・BGM再生ボタンが初回インタラクションとして機能）
- ローカルファイルを `file://` で開いた場合、一部ブラウザでセキュリティ制限が適用される可能性がある（GitHub Pages 経由推奨）
- 背景画像・BGM・タイマー音のファイルはリロードで消去される（設定の永続化は未実装）

## デスクトップアプリ版（Mac）

Electron を使用した Mac 向けデスクトップアプリ版を提供。

### ブラウザ版との違い

| 項目 | ブラウザ版 | デスクトップ版 |
|------|-----------|--------------|
| タブを閉じると停止 | ✕ | ✅ ウィンドウを閉じてもタスクトレイで動作 |
| PCスリープで停止 | △ | ✅ スリープ復帰後も継続 |
| 自動再生制限 | あり | なし |

### 配布ファイル

| ファイル | 対象 |
|---------|------|
| `Audio Timer-1.0.0-arm64.dmg` | Apple Silicon Mac（M1/M2/M3/M4）|
| `Audio Timer-1.0.0.dmg` | Intel Mac |

### インストール手順

1. お使いの Mac に合った `.dmg` をダブルクリック
2. `Audio Timer` を `アプリケーション` フォルダにドラッグ
3. アプリケーションフォルダから起動

> **初回起動時の注意:** 「開発元を確認できません」と表示された場合は、システム設定 → プライバシーとセキュリティ → 「このまま開く」をクリック

### デスクトップ版ファイル構成

```
audio-timer-desktop/
├── main.js            # Electron メインプロセス
├── audio-timer.html   # アプリ本体（ブラウザ版と共通）
├── package.json
└── dist/
    ├── Audio Timer-1.0.0-arm64.dmg   # Apple Silicon 向け
    └── Audio Timer-1.0.0.dmg         # Intel Mac 向け
```
